# Interactive EU Tourism GISN Explorer

This repository contains an interactive visualisation of the
Governance-Integrated Sustainability Networks (GISN) identified in
20 European Union national tourism strategies.

## Interactive visualisation

The interactive explorer is available at:

https://mfuster3.github.io/eu-tourism-gisn/

Users can select a country, inspect the position of the 15
sustainability themes, view node and edge attributes, and access the
official webpage and strategy document where available.

## How to interpret the networks

- Node colour indicates the GISN category.
- Node size represents Weighted Degree.
- Edge width and colour represent the number of coded thematic
  connections between each pair of sustainability themes.
- Identical node positions and visual scales are used across all
  country networks.

## Repository contents

- `index.html`: interactive web visualisation.
- `GISN_interactive_explorer.py`: reproducible Python script used to
  generate the visualisation.
